# Boss Come (老板来了) 🚀

一个功能强大的 Flutter 悬浮球「一键黑屏/摸鱼」神器。支持动态尺寸、透明度调节及系统级悬浮窗覆盖。

---

## ✨ 项目特色
- **极速响应**：点击悬浮球，瞬间启动全屏覆盖，告别尴尬。
- **高度自定义**：自由调节球体大小、颜色、透明度。
- **安全免责**：内置使用权限管理，仅用于技术交流。
- **轻量无感知**：占用内存极低。
